## **Como escribir un "Hello World" o "Hola mundo", en tres lenguajes: JavaScript, Python y Java.**

### Python  
```print("Hola Mundo")```

### JavaScript  
```console.log("Hola Mundo");```

### Java 
```
 public class Main {
    public static void main(String[] args) {
        System.out.println("Hola Mundo");
    }
}
```
## **¿Qué hace el programa Hello World?**  
Verifica que el entorno de desarrollo está bien configurado. Si el mensaje aparece, significa que el compilador o intérprete está leyendo el archivo, la memoria está funcionando y la salida estándar está conectada.

## **¿Qué diferencias visuales notan entre JavaScript, Python y Java?**  
**Python:** Es minimalista y directo. Se enfoca en la acción.  
**JavaScript:** Necesita una referencia clara de dónde estás imprimiendo y suele usar el punto y coma (;) para marcar el fin del ciclo.  
**Java:** Es mas complejo porque requiere que definas el contexto antes de ejecutar la acción. Necesitas definir la estructura de la máquina (class), el sistema de energía (static void main) y luego el actuador (System.out.println). 

## **¿Qué creen que significa ‘sintaxis’ después de ver estos ejemplos?**  
En programación, la sintaxis es el conjunto de reglas que definen cómo se deben combinar los símbolos y palabras para que la computadora entienda la instrucción.
